# Custom Views Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![CustomView](https://img.shields.io/badge/Component-Custom_View-orange.svg)](https://developer.android.com/develop/ui/views/custom-view)

Тестовое приложение для изучения создания кастомных View в Android. Демонстрирует создание переиспользуемого компонента с кастомизацией и сохранением состояния.

## Функциональность

- Кастомная View с двумя кнопками и прогресс-баром
- Настройка через атрибуты XML (цвета, текст)
- Обработка кликов с колбэками
- Режим прогресса с автоматическим отключением
- Сохранение и восстановление состояния

## Технологии

- Kotlin
- Custom View
- View Binding
- Parcelable state
- Custom attributes
- Handler для отложенных задач
